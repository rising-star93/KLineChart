## Installing
### Using npm
```bash
npm install klinecharts --save
```

### Using yarn
```bash
yarn add klinecharts
```

## Build
Execute command in root directory. [Node.js](https://nodejs.org) is required.
```bash
# Install the dependencies from NPM:
npm install

# Build files:
npm run build
```
The generated files are in the dist folder.
